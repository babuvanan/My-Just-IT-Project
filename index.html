Project 1: YouTube Clone

YouTube's homepage

Coding Language: HTML and CSS

Description: This project involved creating a simplified clone of YouTube using HTML and CSS. The goal was to replicate the layout and design of YouTube's homepage, focusing on the visual aspects and responsiveness.

Features:

  • Responsive Design: Adjusts to different screen sizes, ensuring a good user experience on both desktop and mobile devices.
  • Video Thumbnails: Displays a grid of video thumbnails similar to YouTube's layout, each linking to an actual video page.
  • Navigation Bar: Includes a header with a search bar and navigation links.

Project 2: Rock Paper Scissors Game

Rock Paper Scissors Game

Coding Language: HTML, CSS, JavaScript

Description: This project is a simple Rock Paper Scissors game implemented using HTML, CSS, and JavaScript. The game allows the user to play against the computer, which randomly selects its move.

Features:

  • Interactive Gameplay: Users can select rock, paper, or scissors to play against the computer.
  • Random Computer Moves: The computer randomly chooses its move each round.
  • Score Tracking: Displays the score of the player and the computer.
  • Welcome Message: Displays a personalized welcome message with the player's name when they enter their name and press enter.
  • Timer: A 30-second timer starts when the user presses enter.
  • Voice Messages: Results are announced using voice messages.

Project 3: Breakdown Company Database

Breakdown Company Database

Coding Language: MySQL

Description: This project involved designing and implementing a database to model data for a breakdown company. The database handles information about vehicles, customers, and service requests.

Features:

  • Customer Management: Stores customer name and location.
  • Vehicle Management: Records information about vehicles including make, model, and registration number.
  • Breakdown Tracking: Logs breakdown events with details about the location, date, time, and the assigned engineer.
  • Engineer Management: Manages information about engineers, including their contact details and assigned tasks.
  • Data Integrity: Uses foreign keys to maintain data integrity between related tables.
  • Sample Data: Inserts sample data to demonstrate the database functionality.

Project 4: FilmFlix Film Database Application

FilmFlix Film Database

Coding Language: Python

Description: This project is an application developed in Python for managing a movie database for FilmFlix. The application allows for adding, updating, deleting, and viewing movies in the database.

Features:

  • Add movies: Allows users to add new movies to the database.
  • Update movies: Provides functionality to update details of existing movies.
  • Delete movies: Enables the removal of movies from the database.
  • View movies: Lists all movies with details like title, release year, rating, duration and genre.
